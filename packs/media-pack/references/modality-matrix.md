# Media Pack Modality Matrix

## Image

- Typical outputs: hero image, thumbnail, resized variant, transparent cutout
- Key constraints: dimensions, color profile, file size, background treatment
- Review focus: sharpness, crop safety, brand consistency, text legibility

## Video

- Typical outputs: primary cut, teaser cut, subtitle burn-in variant, alternate aspect ratio
- Key constraints: duration, fps, codec, bitrate, loudness, caption policy
- Review focus: pacing, framing, audio sync, export compatibility, disclosure needs

## Audio

- Typical outputs: voiceover, music bed, cleaned interview clip, short sting
- Key constraints: duration, sample rate, loudness, rights, voice policy
- Review focus: clipping, intelligibility, room noise, licensing, cue alignment

## Adaptation

- Typical outputs: resized asset, reformatted export, localized caption pack, channel-specific cut
- Key constraints: platform format, publish slot, naming, packaging, turnaround time
- Review focus: derivative fidelity, consistency with master, channel-specific acceptance
