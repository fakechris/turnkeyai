# Example Request

Use `Media Pack` when you need media deliverables with explicit production and review constraints.

Example:

> Build a launch media package for a new product demo.
> Audience or consumer: B2B buyers on X and LinkedIn.
> Required outputs: one 30-second landscape video, one square teaser cut, three static hero images.
> Source assets or references: product screenshots, brand guide, prior campaign examples.
> Constraints: keep to brand colors, no synthetic voice, max 50 MB per video, export social-ready files.
> Acceptance bar: every file is publish-ready, named consistently, and traced back to its source or prompt chain.
